# Portfolio Website

A clean, modern, static personal portfolio website built using only **HTML5** and **CSS3**. No JavaScript or external frameworks were used.

## Features
- **Responsive Design**: Mobile-first approach using Flexbox and CSS Grid.
- **Pages**: Home, About, Skills, Projects, Contact.
- **PWA Support**: Includes `manifest.json` and a fallback `offline.html`.
- **Clean Code**: Semantic HTML and commented CSS.

## Project Structure
```
portfolio-website/
├── index.html
├── about.html
├── skills.html
├── projects.html
├── contact.html
├── offline.html
├── manifest.json
├── css/
│   └── style.css
└── assets/
    ├── images/
    └── icons/
```

## How to Run
1. Unzip the file.
2. Open `index.html` in any modern web browser.

## Customization
- **Theme Color**: Modify `--primary-color` in `css/style.css`.
- **Content**: Edit the `.html` files to add your own information.
